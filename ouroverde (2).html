📋 Como achar as informações abaixo: 1. console.firebase.google.com → seu projeto ouroverde2702
2. Clique na engrenagem ⚙ → Configurações do projeto
3. Role até "Seus apps" → clique no app web → veja o firebaseConfig
4. Copie o valor de apiKey e o projectId e cole abaixo
⚠ Precisa liberar as Regras do Firestore: 1. No Firebase Console → Firestore Database → aba Regras
2. Apague tudo que tiver e cole exatamente:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } } 3. Clique em Publicar e tente novamente.
